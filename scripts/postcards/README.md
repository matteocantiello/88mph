# Postcard Image Generation

Generate "sonic time travel" postcard images for each chart entry using AI image generation.

## Pipeline

```
chart JSON files  →  prompts file  →  images
data/charts/         data/postcard-prompts.jsonl   public/postcards/
```

### Step 1: Generate Prompts

Extract chart contexts and create image prompts:

```bash
node scripts/postcards/generate-prompts.mjs
```

This creates `data/postcard-prompts.jsonl` — one JSON line per entry with `country`, `year`, and `prompt` fields.

**The prompts file is meant to be edited.** The script preserves existing entries and only adds new ones for charts that don't have a prompt yet. To regenerate all prompts from scratch, delete the file first.

For higher-quality prompts, you can feed `data/chart-contexts.txt` (generated by `scripts/extract-contexts.mjs`) to an LLM and replace the prompts in the JSONL file.

Options:
- `--country us` — only process one country
- `--year 1985` — only process one year

### Step 2: Generate Images

Generate postcard images via Together AI's FLUX models:

```bash
node scripts/postcards/generate-images.mjs
```

Requires `TOGETHER_API_KEY` in `.env.local` or environment.

Output: `public/postcards/{country}_{year}.webp`

Options:

| Flag | Default | Description |
|------|---------|-------------|
| `--country us` | all | Filter by country |
| `--year 1985` | all | Filter by year |
| `--model <id>` | `FLUX.1-schnell` | Together AI model ID |
| `--width 1280` | 1280 | Image width in pixels |
| `--height 736` | 720 | Image height in pixels (must be multiple of 32 for FLUX pro models) |
| `--steps 4` | 4 | Diffusion steps (more = better quality, slower) |
| `--seed 42` | random | Reproducible output |
| `--prefix "..."` | none | Prepend to all prompts |
| `--delay 1000` | 1000 | Delay between API calls (ms) |
| `--force` | false | Regenerate existing images |
| `--dry-run` | false | Preview without calling API |

### Models

Available FLUX models on Together AI (from cheapest to best):

| Model ID | Quality | Cost |
|----------|---------|------|
| `black-forest-labs/FLUX.1-schnell-Free` | Good, fast | Free |
| `black-forest-labs/FLUX.1-schnell` | Good, fast | $ |
| `black-forest-labs/FLUX.1.1-pro` | High quality | $$ |
| `black-forest-labs/FLUX.2-pro` | Very high quality | $$$ |
| `black-forest-labs/FLUX.2-max` | Best quality | $$$$ |

Recommended: Start with `FLUX.1-schnell-Free` to test, then use `FLUX.1.1-pro` or `FLUX.2-pro` for final images.

## Examples

```bash
# Test with one entry (free)
node scripts/postcards/generate-images.mjs --country us --year 1985 --dry-run
node scripts/postcards/generate-images.mjs --country us --year 1985

# Generate all with the free model
node scripts/postcards/generate-images.mjs

# Regenerate all with a better model
node scripts/postcards/generate-images.mjs --force --model black-forest-labs/FLUX.1.1-pro --steps 20

# Higher resolution
node scripts/postcards/generate-images.mjs --width 1920 --height 1080 --model black-forest-labs/FLUX.2-pro
```

## Adding New Charts

When new country/year charts are added:

1. Run `node scripts/extract-contexts.mjs` to update `data/chart-contexts.txt`
2. Run `node scripts/postcards/generate-prompts.mjs` to add new prompt entries
3. Edit `data/postcard-prompts.jsonl` to refine the new prompts if desired
4. Run `node scripts/postcards/generate-images.mjs` to generate only the new images
